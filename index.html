Aura Flowers

Desenvolvi um site floricultura AuraFlowers, utilizando exclusivamente HTML5 e CSS3. Api DevForge

Sistema web para gestão de atestados médicos e avaliação de equipes Scrum, utilizando HTML, CSS, Bootstrap e JavaScript no front-end, Python com Flask no back-end. Fui responsável por desenvolver a funcionalidade no back-end que exibe o atestado em PDF, além de implementar os botões de aceitar ou recusar o documento. Sistema de Hamburgueria

Este projeto foi desenvolvido em linguagem C e consiste em um sistema de pedidos de hambúrgueres. A aplicação permite registrar até quatro pedidos, armazenando também o sexo de cada cliente. Após o registro, o sistema realiza cálculos estatísticos com base nas informações fornecidas, como a quantidade de pedidos por sexo e o total de hambúrgueres vendidos. Ao final, um menu interativo é exibido para que o usuário possa consultar detalhes específicos dos pedidos e visualizar os dados de forma organizada. Esse projeto foi importante para exercitar lógica de programação, uso de estruturas de repetição, condicionais e manipulação de dados em linguagem C.
